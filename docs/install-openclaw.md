# Install in OpenClaw

OpenClaw skill layout may vary by installation. The portable approach is:

1. Locate your OpenClaw skills directory.
2. Copy `skills/prompt-dojo/` into that directory.
3. Start a fresh OpenClaw session.
4. Ask for a Prompt Dojo lesson.

Example shape:

```bash
cp -R skills/prompt-dojo /path/to/openclaw/skills/prompt-dojo
```

Keep this repo as the canonical public source. Runtime-specific paths should stay in install docs, not in the skill body.
