# Basic Prompt Template

```text
Task:
[What I want done]

Context:
[What matters, who it is for, what the model must know]

Constraints:
[What not to do, boundaries, tools, style, safety]

Output:
[Exact format I want]

Quality bar:
[How I know the answer worked]
```

## Use when

- the model keeps guessing
- output is too broad
- you need repeatable prompt quality
- you are asking an agent to do work with files, tools, or public-facing content

## Common additions

```text
Inputs:
[Paste or attach data here]

Assumptions:
[What the model may assume if something is missing]

Verification:
[How to check the result]
```
